# 📸 Onde colocar as fotos

Salve as imagens dentro da pasta **`public/`** com EXATAMENTE estes nomes.
Pode soltar quando quiser — enquanto a foto não existir, aparece um placeholder
elegante no lugar (nada quebra). Assim que o arquivo estiver na pasta, ela aparece.

Formatos: **.jpg** (de preferência). Se usar .png/.webp, me avise pra ajustar a extensão.
Dica de tamanho: ~1200px no lado maior já fica ótimo (o site otimiza sozinho).

---

## 3) Colagem "Novidades"  → pasta `public/colecao/`
- `colecao/01.jpg` — vertical (a maior)
- `colecao/02.jpg` — quadrada
- `colecao/03.jpg` — vertical (menor)

## 4) Comprar por ocasião  → pasta `public/ocasioes/`
Verticais, com clima/pessoa usando a joia se possível:
- `ocasioes/presente.jpg`
- `ocasioes/namoro.jpg`
- `ocasioes/masculino.jpg`
- `ocasioes/parami.jpg`

## 5) Compartilhamento no WhatsApp/Instagram (OpenGraph)
- `public/og.jpg` — horizontal 1200×630. É a imagem que aparece quando o link é
  compartilhado. Use uma foto bonita de peça.

---

**Resumo rápido:** é só arrastar os arquivos pra dentro de `public/` (e subpastas)
com esses nomes. Não precisa mexer em código. Se quiser usar outros nomes, me manda
a lista que eu ligo certinho.
